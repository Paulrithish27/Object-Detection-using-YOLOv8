# Object-Detection-using-YOLOv8
This project demonstrates object detection using the YOLOv8 model.
 The model detects objects in an image and draws bounding boxes with class labels and confidence scores.

 YOLO (You Only Look Once) is a fast and accurate deep learning model used for real-time object detection.
 
---
#Features

Detect objects in images

Draw bounding boxes around detected objects

Display object class names and confidence scores

Save the output image with detections
